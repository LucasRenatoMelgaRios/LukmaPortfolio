# Portfolio de Lucas Melgar

Portfolio personal construido con Astro, React, Tailwind CSS y TypeScript.

## Desarrollo

```bash
bun install
bun run dev
```

## Producción

```bash
bun run build
bun run preview
```

La página principal vive en `src/pages/index.astro`. Los visuales conceptuales de los proyectos están en `src/components/ProjectVisual.astro` y pueden sustituirse por capturas reales cuando estén disponibles.
